# Assets

This directory contains app icons and splash screens.

Required files:
- icon.png (1024x1024)
- splash.png (2048x2048)
- adaptive-icon.png (1024x1024)
- favicon.png (48x48)

You can generate these using:
- Online tools (e.g., AppIcon.co)
- Design software (Figma, Sketch)
- Or use placeholder images for development
